# Быстрый старт

## Запуск приложения за 3 шага

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустите приложение
```bash
python app.py
```

### 3. Откройте в браузере
```
http://localhost:5000
```

## Первый вход администратора

1. На главной странице нажмите "Вход администратора"
2. Введите код: `oversun2025` (можно изменить в `config.py`)

## Тестирование

### Регистрация участника:
1. Нажмите "Принять участие"
2. Заполните форму:
   - Имя: Иван Иванов
   - Email: ivan@oversun.ru
   - Пожелания: Книга о программировании
3. Нажмите "Зарегистрироваться"

### Запуск жеребьёвки:
1. Войдите как администратор
2. Зарегистрируйте минимум 2 участников
3. Нажмите "Запустить жеребьёвку"
4. Участники увидят своих получателей в личном кабинете

## Важно

- База данных создается автоматически при первом запуске
- Код администратора можно изменить в `config.py`
- Для продакшена измените `SECRET_KEY` в `config.py`
